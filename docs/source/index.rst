Welcome to PYIIDES documentation!
=================================

PyIIDES is a Python implementation of the `Insider Incident Data Exchange Standard (IIDES) <https://github.com/cmu-sei/iides>`_  schema. This package provides tools for the coding, storage, and sharing of data related specifically to insider incidents of all kinds.
